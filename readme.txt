Documentation for building, running, and testing the program.

0) Pre-requisites:

- Apache Maven 3.3.9
- JDK 8

1) Build 

- Extract the attached zip file 
- cd expression_calculator
- mvn clean package

2) Run (Windows)

- Edit the file run.bat to enter the expression to be evaluated (EXPRESSION variable)
- Please enclose the expression in double quotes if there are blank spaces within the expression. This is only necessary to delimit the argument passed to the JVM. 
- Execute run.bat
